Bootsrap

Bootstrap is a free front-end framework (HTML and CSS) for faster and easier web development. Bootstrap is the most popular CSS Framework for developing responsive and mobile-first websites.

What is Responsive Design?

  • Responsive design focuses on showing content on the basis of available browser space.
  • When a website developed using responsive design is opened on a desktop browser and when we try to resize the browser window, the content of the website is dynamically and optimally rearranged to accomodate the window.
